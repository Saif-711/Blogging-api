# Blogging API Project

## Project URL
[Project Link](https://github.com/Saif-711/Blogging-api)

## Overview
A Spring Boot blogging application that allows users to create, read, update, and delete blog posts. The application also supports categories and tags for better content organization.

## Features
- Create, read, update, delete blog posts
- Manage categories and tags
- Automatic timestamps for `createdAt` and `updatedAt`
- Input validation for title, content, and category

## Technology Stack
- Java 17
- Spring Boot
- Hibernate / JPA
- Lombok
- PostgreSQL / MySQL (or H2 for testing)
